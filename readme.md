# 🐟 Fiskodlingsassistent med AI

En AI-assistent som hjälper odlare att välja rätt fiskarter baserat på vattenparametrar.

## 📌 Funktioner
- Importerar vattendata från CSV/Excel.
- Validerar att frågor är relevanta och säkra.
- Ger rekommendationer om fiskodling med OpenAI/Mistral.
- Sparar historik och feedback för framtida förbättringar.
